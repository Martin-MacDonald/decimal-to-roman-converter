# Decimal to Roman Numerals Converter
  > Convert any decimal integer from 1 - 3999 to the equivalent roman numeral 

## Technologies Utilised
  - Node v10.15.0
  - npm v6.4.1
  - jest

## Setup
Install packages with [npm](https://www.npmjs.com/):
```sh
$ npm install
```

## Running Tests
[See test definition](./test.js)
```sh
$ npm test
```
